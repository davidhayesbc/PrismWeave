🚀 Quick Start:
   npm install && npm run build && npm link
   prismweave config --set githubToken=TOKEN
   prismweave config --set githubRepo=owner/repo
   prismweave capture https://example.com

✨ Features:
   ✓ Headless browser capture with Puppeteer
   ✓ Clean HTML to Markdown conversion
   ✓ Automatic GitHub commits
   ✓ Smart file organization
   ✓ Batch URL processing
   ✓ 60% code sharing with browser extension

🎯 Usage Examples:
   # Single URL capture
   prismweave capture https://blog.com/article

   # Batch processing
   prismweave capture --file urls.txt

   # Preview without saving
   prismweave capture URL --dry-run

   # Custom timeout for slow sites
   prismweave capture URL --timeout 60000

   # Custom commit message
   prismweave capture URL --message "Great article"

🔧 Configuration:
   prismweave config --set githubToken=TOKEN
   prismweave config --set githubRepo=owner/repo
   prismweave config --test
   prismweave config --list

📦 Requirements:
   • Node.js 18.0.0 or higher
   • GitHub personal access token
   • Internet connection
