## 🛠️ Setup Resources

### Automated Setup

#### **setup.sh** - Unix/Linux/Mac Setup Script
One-command setup for Unix-like systems.

**What it does:**
- ✅ Checks Node.js installation
- 📦 Installs dependencies
- 🔨 Builds the project
- 🔗 Links globally (optional)
- ⚙️ Configuration wizard

**Usage:**
```bash
chmod +x setup.sh
./setup.sh
```
