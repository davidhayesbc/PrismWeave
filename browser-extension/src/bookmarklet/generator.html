PrismWeave Bookmarklet Generator

Create a personalized bookmarklet with your GitHub settings embedded for universal content capture across all domains.

What is this?

This tool generates a custom bookmarklet that includes your GitHub Personal Access Token and repository settings directly in the code. This eliminates the need for cross-domain storage and ensures your bookmarklet works on any website without asking for credentials repeatedly.

🔒 Security Note

Your GitHub token will be embedded in the bookmarklet code. Only share the generated bookmarklet with trusted parties, and consider using a token with limited repository access.
